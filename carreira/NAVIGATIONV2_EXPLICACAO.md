# navigationV2

**O que é:** Aplicativo mobile para coleta de dados em campo.

**Problema que resolve:**
- Elimina papel e planilhas na coleta de solo.
- Garante validação, rastreabilidade e qualidade dos dados.
- Sincronização automática com AvanteHub.
- Controle de sessões, motivos de pulo, exigência de foto.

**Stack:** TypeScript, React Native, Expo, Lucide, Animated, Supabase.

**Principais módulos:**
- Interface para técnicos.
- Validação de entrada.
- Processamento e envio de dados.
- Integração direta com AvanteHub via API.

**Público-alvo:** Técnicos de campo, empresas do agro, gestores.
