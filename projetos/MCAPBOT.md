# MCapBot

## O que é
Bot de automação de rotinas fiscais para escritório de advocacia. Automatiza interações com e-CAC e ReceitaNet da Receita Federal.

## Problema que resolve
Advogados e contadores gastavam horas em tarefas repetitivas no portal e-CAC e ReceitaNet. MCapBot automatiza essas rotinas, reduzindo trabalho manual e erros humanos.

## Como funciona
1. Bot acessa portais fiscais (e-CAC, ReceitaNet) automaticamente
2. Executa rotinas pré-configuradas (consultas, downloads, preenchimentos)
3. Extrai dados e gera relatórios
4. Interface web simples para configuração e acompanhamento

## Arquitetura
- **Backend:** Python (automação/scraping)
- **Frontend:** HTML (interface de controle)
- **Automação:** Selenium/requests para interação com portais governamentais
