# Site Institucional MCADV

## O que é
Site institucional do escritório Moraes de Carvalho Advogados Associados. Passou por 3 iterações baseadas em feedback dos sócios até a versão premium final.

## Problema que resolve
Escritório precisava de presença digital profissional. Site apresenta o escritório, áreas de atuação e informações de contato.

## Iterações
1. **v1 (Mcap)** — Primeira versão, JavaScript básico (Mai/2024)
2. **v2 (mcap-2-oficial / mcap-3-oficial)** — Refinamentos com feedback (Jun/2024)
3. **v3 (mcadv.site)** — Versão premium final com design profissional (Jun/2024, atualizado até Fev/2026)

## Arquitetura (versão final)
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP (formulário de contato)
- **Estilização:** CSS customizado (79KB)
- **Deploy:** Hospedagem própria
